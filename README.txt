Godot 4.4 — Hex-Ready Minimal Project

1. Import `project.godot` in Godot 4.4.
2. Open scene `src/scenes/world/World.tscn`.
3. Select `TileMapLayer` → Paint tool → click in viewport to place hex tiles.
4. Camera: WASD/arrow keys to move, mouse wheel to zoom.

Settings in TileSet:
- Tile Shape = Hexagon
- Tile Size = 64×64
- Hex Side Length = 32
